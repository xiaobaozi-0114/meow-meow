# Meow Meow

## 立即打开

**网页入口： [https://xiaobaozi-0114.github.io/meow-meow/](https://xiaobaozi-0114.github.io/meow-meow/)**

Meow Meow 是一份面向中国养猫家庭的猫咪百科与养猫助手。无论你是刚准备接猫的新手，还是已经养猫多年、想快速查资料的铲屎官，都可以在这里找到品种资料、喂养护理、用品参考、异常症状分诊和智能问答。

## 你可以用它做什么

- 查猫咪品种：了解常见猫咪的外观、性格、养护难度、适合人群和健康关注点。
- 学新手养猫：从接猫前准备、到家适应、喂食换粮、疫苗驱虫、绝育、清洁护理一步步上手。
- 看养猫技巧：覆盖饮食、猫砂盆、清洁、互动、环境布置、行为问题和日常观察。
- 参考猫咪用品：整理猫粮、猫罐头、猫条、日常保健品和生活用品的品牌信息、价格区间、常见优缺点与注意事项。
- 做紧急分诊：遇到呕吐、腹泻、尿闭、精神差、误食等情况时，快速判断风险级别和下一步处理方向。
- 问智能助手：网页没有覆盖的问题，可以直接向智能养猫助手提问。

## 推荐使用方式

1. 第一次养猫，先看“新手教程”，按时间线准备用品、食物、体检和居家环境。
2. 想选猫或了解猫咪性格，进入“品种大全”，对比不同品种的毛长、体型、性格和护理难度。
3. 日常遇到小问题，先搜关键词，比如“换粮”“软便”“猫砂”“刷牙”“封窗”。
4. 想买东西前，去“猫咪用品”查看常见品牌的优缺点和注意事项。
5. 猫咪突然不对劲，优先打开“生病自救”，先判断是否需要立刻就医。
6. 资料里没有写清楚的问题，直接问“智能养猫助手”。

## 适合谁

- 准备第一次养猫的人
- 刚接猫、不知道每天该做什么的新手
- 想系统补齐养猫知识的铲屎官
- 想快速对比猫咪品种和用品的人
- 遇到猫咪异常症状、需要先判断轻重缓急的人

## 使用提醒

Meow Meow 适合作为养猫知识参考和日常决策辅助。涉及急症、用药、剂量、诊断、处方、慢性病、幼猫、孕猫、老年猫或基础病猫时，请及时联系兽医或动物医院。

猫咪如果出现尿不出、持续呕吐、明显呼吸异常、抽搐、误食有毒物、严重外伤、精神极差、体温异常等情况，请不要等待网页建议，直接就医。
